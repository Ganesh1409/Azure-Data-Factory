# Azure-Data-Factory
ADF pipelines for data movement from onpremise to ADLS Gen2, Blob storage to ADLS Gen2 and vice versa
Azure data factory is a cloud based ETL tool which is used for Data Ingestion and Archestration purpose.
It is mostly preferred for Data Ingestion from difference sources to the cloud.
It is a non coding environment still we can see the backend code in Json format.
